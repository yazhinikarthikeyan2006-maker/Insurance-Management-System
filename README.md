# 🛡️ Insurance Management System

A Full Stack Insurance Management System developed using **HTML, CSS, JavaScript, Node.js, Express.js, MySQL, and JWT Authentication**.

This project allows users to register, login securely, and manage insurance policies through a professional dashboard.

---

## 🚀 Features

- 👤 User Registration
- 🔐 User Login using JWT Authentication
- 🏠 Professional Dashboard
- ➕ Add Insurance Policy
- 📋 View All Policies
- ✏️ Edit Policy
- 🗑️ Delete Policy
- 📊 Dashboard Statistics
- 💾 MySQL Database Integration

---

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MySQL

### Authentication
- JSON Web Token (JWT)
- bcrypt

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yazhinikarthikeyan2006-maker/Insurance-Management-System.git
```

### Install Packages

```bash
npm install
```

### Run Server

```bash
node server.js
```

Server will start at:

```
http://localhost:3000
```
